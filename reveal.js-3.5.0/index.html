Développeur: Un développeur conçoit ou améliore un programme informatique/logiciel.

Développeur front-end: Un développeur front-end est celui qui s'occupe de toute la partie visuel d'un site web.
Développeur back-end: Un développeur back-end est celui qui va s'occuper de toutes les fonctionnalités d'un site web qui sont aussi la partie non visible d'un site web.
Développeur full-stack: On peut le considérer comme un développeur complet. Il peut aussi bien développer la partie front que la partie back d'un site web.
